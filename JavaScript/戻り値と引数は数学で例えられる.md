### 戻り値はyに例えられる

戻り値とは、数学におけるy=2xのyのようなもの。

具体的には、何か計算した結果を最終的に返すのが戻り値の役割。

### 引数はx

引数は、y=2xにおける「x」のようなもの。

**具体例**
```
function hikisu (a) {
 const test = a + 1;
 return test;

}

const result = hikisu(100);
console.log(result);
```

コンソール画面の出力結果
```
101
```
